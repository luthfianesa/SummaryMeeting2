__Sejarah singkat git__

  • Linus Torvalds (Pencipta Linux) membuat project linux menjadi open source.
  • Karena banyak developer yang ikutan, Akhirnya menimbulkan konflik antar developer (Kodenya berubah-berubah)
  • Oleh karena itu Linus membuat Git
  • Git berfungsi untuk mengontrol dan mendokumentasikan kode dari project yang dikerjakan

Cara cek versi git -> git --version | My current git version 2.37.2

For 1st timer, setup git

Ketik di Terminal :

git config --global user.email "you@example.com"

git config --global user.name "Your Name"



__Upload file ke github__

Masuk ke dalam terminal git bash, lalu ketik perintah :

- git init (Untuk membuat repository pada file lokal yang nantinya ada folder .git)

- git remote add origin git@github.com:usernamenya/namaRepositorynya.git (Menghubungkan repository lokal dengan repository remote)

- git status (Untuk mengetahui status dari repository lokal)

- git add . (Menambahkan semua file baru pada repository yang dipilih)

- git commit -m 'isi dengan pesan'(Untuk menyimpan perubahan yang dilakukan, tetapi tidak ada perubahan pada remote repository)

- git push origin namabranchnya (Push branch yang telah dibuat ke remote repository)

- Balik lagi ke dalam halaman repository di web github, cek perubahannya



__Perintah git lainnya__

git remote show origin -> Mengetahui link url

git branch -> Menampilkan branch yang sedang aktif

git branch -a -> Buat Menampilkan list branch

git checkout -b "namaBranchnya" -> Membuat branch baru

git checkout namaBranchnya -> Pindah branch yang diinginkan

git fetch -> Mengupdate perubahan yang ada dalam repo

git clone-> Mengkloning repository

Ada 2 cara dalam kloning repository

  1. Menggunakan HTTPS -> git clone https://github.com/usernamenya/namaReponya.git
  2. Menggunakan SSH -> git clone git@github.com:usernamenya/namaReponya.git
